# Codebase Structure and Context Summary

## Root
- app.py: Likely main backend entry point (Python)
- requirements.txt: Python dependencies
- README.md: Project documentation

## Backend (src/)
- __init__.py, error_handlers.py, initialization.py: Core backend setup
- endpoints/: API endpoints (image_detection.py, routes.py)
- models/: Model definitions (cnn_model.py)
- pipeline/: Data/ML pipeline (data_ingestion.py, datasets.py, evaluation.py, feature_engineering.py, inference.py, model_training.py, preprocessing.py)
- scripts/: Automation/setup (download_easyocr_models.py, manual_handwriting_ocr.py, setup_ocr.py, train_cnn_from_scratch.py, web_scraping_fix.py)
- services/: Backend services (recommendation_service.py)
- utils/: Utilities (data_cleaning.py, delete_pinecone_index.py, enhanced_spell_checker.py, handwriting_ocr.py, logging_utils.py, model_loader.py, upload_to_pinecone.py, vector_db_utils.py)
- data/: Datasets, progress, scraped images, label mapping, etc.

## Frontend (frontend/)
- next.config.js, package.json, tailwind.config.ts, etc.: Next.js/React setup
- app/: Next.js app routes (page.tsx, layout.tsx, globals.css, subfolders for text-query, ocr-query, image-upload)
- components/: Reusable React components (loading-spinner, navigation, result-card, ui/ with many UI primitives)
- hooks/: Custom React hooks (use-toast.ts)
- lib/: Frontend utilities (utils.ts)

## Data (src/data/)
- Datasets: cleaned_cnn_data.csv, dataset.csv, product_terms.txt, etc.
- Progress: upload_progress.csv
- Scraped data: www.amazon.com/product_*.jpg
- Test images: test.png

## Notebooks (notebooks/)
- 1_pipeline_integration_test.ipynb
- 2_similarity_analysis.ipynb
- 3_ocr_scraping_utils.ipynb

## Tests (test/)
- test_api_endpoints.py, test_data_cleaning.py, test_easyocr.py, test_enhanced_spell_checker.py, test_handwriting_ocr.py, test_logging_utils.py, test_model_loader.py, test_vector_db_utils.py

## Model Test Images (model_test/)
- Contains subfolders and images for model testing
